
## 1、本地模块

mkdir link-module

index.js

需要的项目里执行： npm link {path}/link-module (默认会装在全局)


## 2、node 命令开发

```json
{
  "name": "npmtest",
  "version": "1.0.0",
  "description": "test",
  "main": "index.js",
  "bin": {
    "hello": "./index.js"
  },
  "scripts": {
    "test": "test"
  },
  "author": "",
  "license": "ISC",
  "dependencies": {},
  "devDependencies": {
    "@babel/core": "^7.4.5"
  }
}
```json

npm link module使bin起作用
